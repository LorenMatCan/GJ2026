# GJ2026 – FeR

## Prerrequisitos

El proyecto fue desarrollado usando **Unity 2019.4.32f1**.  
Se recomienda contar con esta misma versión en caso de querer editar el proyecto dentro de Unity.

---

## Instalación

1. Clonar el repositorio y guardarlo en una carpeta:
   ```sh
   git clone https://github.com/LorenMatCan/GJ2026
   ```
2. Clonar el repositorio guardándolo en una carpeta
   ```sh
   git clone https://github.com/LorenMatCan/GJ2026
   ```
3. Después abre Unity Hub
   ```sh
   Add -> Add project from disk
   ```
4. Selecciona la carpeta donde guardaste el proyecto (Nombre: Pesadillas)
