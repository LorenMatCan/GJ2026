# Alfa 
## Instrucciones
Para ejecutar el juego necesitarán una computadora con unity.
Abra unityhub, dele click  a la sección de agregar nuevo proeyecto. 
Selecióne el archillo "pesadillas".
De le click a abrir.
## Crear el ejecutable 
Para crear el .exe será necesario que abra el proyecto en unity como se describio anteriormente.
Vaya a al menu de file. Esto ahrá que un submenu aparezca seleccióne la opción de Build and Run.
Esto si se esta usando windows, en caso de usar algúna otra distro pasos extras serán necesarios.
## Ejecutar
Una vez creado el .exe solo será necesario que le de click a GJ2026.exe
## Jugar 
Para mover a Fer use las teclas direccionales de izquierda y derecha o las teclas A y D. Para saltar use la tecla espaciadora, FER cuenta con uns mecánica de doble salto, para esto es necesario que vuelva a presionar la tecla espacio. Mientras más presiones la tecla espacio más alto salta FER. Finalmente las máscaras se usan con las flechas de arriba y abajo.
